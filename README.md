# ⚾ 야구 티켓 예매 연습

## 1. 프로젝트 소개

야구 경기 티켓 예매 흐름을 처음부터 끝까지 연습할 수 있는 멀티 페이지 웹 애플리케이션입니다.
날짜 선택부터 결제 완료까지 실제 예매 사이트와 유사한 단계별 흐름으로 구성되어 있습니다.

## 2. 데모

🔗 https://baseballticket.netlify.app/html/home.html

## 3. 주요 기능

- **날짜 선택** — 오늘 기준 4일간의 LG 경기 자동 생성
- **구역 선택** — 내야석(14,000원) / 외야석(8,000원) 선택
- **좌석 선택** — A·B구역 5×5 그리드, 최대 4석 선택 가능
- **가격 확인** — 티켓 총액 + 수수료(3,000원) 표시
- **배송 선택** — 현장수령 / 모바일티켓 / 택배(+3,000원)
- **결제** — 신용카드(랜덤 카드 생성 후 입력 검증, 3회 실패 시 잠금) / 무통장입금
- **완료** — 예약번호 자동 생성 및 예매 정보 요약

## 4. 기술 스택
